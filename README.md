# Love, Life & Likes Dashboard

Interactive Next.js dashboard for the WIA1006/WID3006 Machine Learning group project.

## Run locally

```bash
npm install
npm run dev
```

## Pages

- `/` dashboard
- `/report` printable report draft
- `/slides` printable slide deck
- `/script` video script

## Deployment

This project is ready for Vercel. Railway can also serve it as a Node app.
